## mill/quake

A visual p2pquake API decoder (json API to a web map)

---

### development

```bash
pnpm i && pnpm dev
```

vite hosts to localhost:5173 by default

---

### required for development

1. mapbox API key

> note: put your API key into `.env`
>
> example: `VITE_MAPBOX_GL_ACCESS_TOKEN=pk.eyJ...`
