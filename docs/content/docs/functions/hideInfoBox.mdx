Hides the infoBox component in the UI.

`hideInfoBox()` → `{void}`

Conceals the information box from the user interface when earthquake information is not needed or available.

#### Functionality

- **Visibility Control**: Hides the infoBox component from the user interface
- **UI State Management**: Updates the display state to remove earthquake information

#### Usage Context

This function is typically called when:
- No earthquake data is available to display
- The application needs to clear previous information displays
- Transitioning between different data views or states
