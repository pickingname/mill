title: armIntList

Arms the intensity list, making it visible. Used when the report type includes intensity/points data such as `foreign`, `destination`.

`armIntList()` → `{void}`

#### Source

```js title="components/infoBox/updateIntList.js"
export function armIntList() {
  intListParent.classList.remove("hidden");
}
```

removes `hidden` class from the `intList` element, making it visible.
